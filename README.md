# 헥사 필 (Hexa Fill) - 모바일 헥사 퍼즐 게임


**[헥사 필]**은 다양한 모양의 육각(헥사) 블록 조각들을 빈 육각판에 정확하게 채워 넣는 클래식하고 직관적인 모바일 퍼즐 게임입니다.

## 주요 기능 (Features)

* **직관적인 드래그 앤 드롭:** 누구나 쉽게 배울 수 있는 간단한 터치, 드래그, 드롭 방식의 조작을 제공합니다.
* **회전 없는 논리 퍼즐:** 블록은 회전할 수 없습니다. 오직 주어진 모양 그대로 알맞은 자리를 찾아야 하므로, 순수한 공간 지각 능력과 논리력을 자극합니다.
* **다양한 난이도의 레벨:** Easy, Medium, Hard 챕터로 구분된 수백 개의 스테이지를 제공하여 초보자부터 숙련자까지 모두 즐길 수 있습니다.
* **데일리 퍼즐:** 매일 새롭게 제공되는 '오늘의 퍼즐'에 도전하고 특별한 보상을 획득하세요.
* **힌트 시스템:** 막히는 구간에서 '힌트'를 사용하여 코인을 소모하고 정답 위치를 하나씩 확인할 수 있습니다.
* **미니멀한 아트 스타일:** 눈이 편안한 파스텔톤 색감과 깔끔한 플랫 디자인을 적용했습니다.
* **차분한 사운드:** 집중을 돕는 명상적인 BGM과 만족감을 주는 효과음(SFX)을 제공합니다.

## 사용 기술 (Tech Stack)
